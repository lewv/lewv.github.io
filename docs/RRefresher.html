Distributions

It’s important to know how to use the functions associated with distributions in R as many advanced textbooks do not have t-tables etc. The first letters of each distribution family have the following meaning:

r - random generation, given a number of observations
p - distribution function, given a quantile
q - quantile function, given a probability
d - density, given a quantile
